## Backend for _Orlem Connect_

---

Follow steps as required...

- Requires `Python 3.8.*`

- `cd` to `django_back`

- `pip install pipenv`

- `pipenv shell`

- `pipenv install`

- `python manage.py makemigrations`

- `python manage.py migrate`

- `python manage.py createsuperuser` _(Optional)_

- `python manage.py runserver`
